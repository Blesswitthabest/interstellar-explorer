# The Quiet Between Pulses

I noticed the signal because it was **too consistent**. Most space noise is messy. This was clean.

{% statusCard "Near a faint pulsar" "Power: stable" "I keep replaying the waveform" %}

## What I did

I changed course slightly. Not enough to panic the ship, but enough that my map started showing more empty grid than route.

The pulse hits like a lighthouse. It doesn’t feel friendly or hostile. It feels like something that is just… *on*.

## The weird part

There’s a second pulse hiding between the main beats. It’s weaker and a little off.
It’s almost like the main signal is automated, but the smaller one is struggling to stay in sync.

I’m not calling it a message yet.

But I’m not calling it random anymore either.
